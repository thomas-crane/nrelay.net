A framework for powerful RotMG bots.

nrelay is a console based modular client for Realm of the Mad God built with NodeJS and TypeScript. It was built with flexibility and plugin development in mind, and a major design goal of nrelay is to make writing plugins both rewarding and easy.
Because it is built on NodeJS, integrating other npm modules is as easy as using npm install. nrelay is also packaged with several pre-built plugin components to provide useful functionality right out of the box.
